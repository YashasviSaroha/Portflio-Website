# 🌐 Portfolio Website

A modern and responsive **developer portfolio website** built to showcase my projects, skills, and experience. Designed with performance, SEO, and clean UI in mind.

---

## 🚀 Live Demo

* 🔗 Frontend: https://portflio-website-theta.vercel.app/
* 🔗 GitHub Repo: https://github.com/YashasviSaroha/Portflio-Website

---

## 🛠️ Tech Stack

- **Frontend:** Next.js
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Version Control:** Git & GitHub

---

## ✨ Features

- 📱 Fully responsive design (Mobile + Desktop)
- ⚡ Optimized performance with Next.js
- 🎨 Modern UI using Tailwind CSS
- 🔍 SEO-friendly structure
- 📂 Projects showcase section
- 📄 Resume download option
- 📬 Contact section

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YashasviSaroha/Portflio-Website.git
cd Portflio-Website
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

* Open 👉 http://localhost:3000

---

## 🔐 Environment Variables

Create a `.env.local` file in the root folder and add:

```
NEXT_PUBLIC_EMAIL=your_email_here
NEXT_PUBLIC_LINKEDIN=your_linkedin_url
NEXT_PUBLIC_GITHUB=your_github_url
```

---

## 🚀 Deployment (Vercel)

- 1. Push your code to GitHub
- 2. Go to **Vercel**
- 3. Import your repository
- 4. Add environment variables
- 5. Click **Deploy**
